# LeetCode Repository

This repository contains all the exercises I solved on [LeetCode](https://leetcode.com/).


### Why

The goal is to keep track of my progress, archive my own solutions, and revisit problems later when needed.

Some exercises also include an alternative_solutions/ folder containing interesting solutions from other people.

I use it to study different approaches, compare optimizations, and archive useful ideas or techniques.

### Structure
```bash
exercise-name/
├── my_solution.c
└── alternative_solutions/
    ├── solution_1.c
    └── solution_2.c
```
